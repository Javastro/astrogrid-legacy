Map of known ACR components / services.
Schema
Element <acrService>
Describes a single acr service.
Attribute id (required) Translator: qualified-id
The service id of the component
Attribute interface (required) Translator: class
The public interface class of the component (may be a superclass of the interface class used in definition of the service)
