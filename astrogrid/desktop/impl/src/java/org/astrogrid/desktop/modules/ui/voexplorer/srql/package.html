Simple Registry Query Language

The package implements SRQL - a simple registry query language. Syntax is quite free-form and inspired by google query language. There's a parser {@link org.astrogrid.desktop.modules.ui.voexplorer.srql.SRQLParser} that returns parsed phrases of the SRQL language - which will be a subtype of the {@link org.astrogrid.desktop.modules.ui.voexplorer.srql.SRQL} base class. The SRQL terms implement the visitor pattern, to allow XQueries, and other things, to be generated from them. See {@link org.astrogrid.desktop.modules.ui.voexplorer.srql.SRQLVisitor} and {@link org.astrogrid.desktop.modules.ui.voexplorer.srql.Builder}
