Astronomy Client Runtime - Implementation

The framework packages contain the classes that assemble the workbench by parsing module descriptors and instantiating modules.

The system module packages implement the basic components - e.g. shutdown, plus the access methods into the functions of the ACR - at the moment this is XMLRPC, RMI, HTTP and GUI.

The vo modules packages implement components that interface with remote virtual observatory services

The ui modules packages implement components that are predominantly visual - dialogues and applications.
