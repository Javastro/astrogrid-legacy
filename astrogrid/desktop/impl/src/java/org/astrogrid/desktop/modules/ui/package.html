User Interface Components, and base classes

This package contains the implementation of the mini-applications of the workbench.

All mini-applications extend from {@link org.astrogrid.desktop.modules.ui.UIComponent}, and perform requests to VO services and other long-running operations in {@link org.astrogrid.desktop.modules.ui.BackgroundWorker} instances. This ensures that : there's a consistent look-n-feel; the UI remains responsive; and improvements to these base classes benefit all the mini-applications.
