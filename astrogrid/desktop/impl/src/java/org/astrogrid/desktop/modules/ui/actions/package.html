UI Activities

Activities are operations that can be performed on the user's current selection - one or more resources (in voexplorer), files (in filemanager), or query results (in astroscope and taskrunner).

Activities appear within the window menus, and also in the Activities Pane, which shows a changing list of activities suitable to the current selection.

Each activity is implemented by a separate class, most of which are in this package. They all must implement the {@link org.astrogrid.desktop.modules.ui.actions.Activity} interface. The baseclass is {@link org.astrogrid.desktop.modules.ui.actions.AbstractActivity}.

Activities are managed by the {@link org.astrogrid.desktop.modules.system.ui.ActivitiesManager}, and assembled using hivemind.
