Welcome to the Workflow viewer ( builder to follow... )

At present you can:
  • load and save workflows from either your MySpace or local file system
  • view workflows in either a tree form or document form
  • submit workflows to the Job Entry System
  • look at details of tasks within the registry
  • search for specific tasks within the registry
Beta changes: drag and drop has been added, this is still to be improved/tested and has no effect on the workflow object you are viewing/creating. This is still basically a workflow viewer at present....
