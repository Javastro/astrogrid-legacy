README
======
This folder contains system and integration tests for AR and Astrogrid services, written as Python Unit Tests (Using the unittest python module)

Developed using Python 2.3.5

use the 'simpletests.py' as a template for adding additional tests.

a reference to tests than can be run unattended should be added to alltests.py

put ui exercising code (i.e not autmatable tests) in the 'experiements' folder instead

put experiements and try-outs in the 'exercises' folder, and omit from alltests.py
