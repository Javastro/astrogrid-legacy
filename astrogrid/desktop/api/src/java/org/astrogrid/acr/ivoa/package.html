Interface to IVOA services and standards

Of most interest are the interfaces to DAL Services -- {@link org.astrogrid.acr.ivoa.Cone}, {@link org.astrogrid.acr.ivoa.Siap}, {@link org.astrogrid.acr.ivoa.Ssap}. These all inherit common functionality from the {@link org.astrogrid.acr.ivoa.Dal}.
For accessing IVOA Resource Registries see {@link org.astrogrid.acr.ivoa.Registry} and {@link org.astrogrid.acr.ivoa.ExternalRegistry}
