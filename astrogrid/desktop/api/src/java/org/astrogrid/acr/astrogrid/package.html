Astrogrid Webservices

This module provides access to Astrogrid-Specific services. Many of these services are fore-runners of upcoming IVOA standards.

Remote File Storage

{@link org.astrogrid.acr.astrogrid.Myspace} is a prototype of the upcoming IVOA VOSpace standard for distributed file systems.

Long-running Remote Applications and Queries

Astrogrid provides a framework for remote execution of complex database queries and long-running processing applications (such as cross-matching codes). This powerful functionality takes a little more effort to access than synchronous data services (such as provided by {@link org.astrogrid.acr.ivoa.Dal} services). It is split between two components
{@link org.astrogrid.acr.astrogrid.Applications}
Handles selecting a remote application, and constructing an invocation
{@link org.astrogrid.acr.astrogrid.RemoteProcessManager}
Execute the application, manage the execution, and retreive the results

Authentication

Sign-on and Authentication for Astrogrid services is provided by the {@link org.astrogrid.acr.astrogrid.Community} component.

Time-Series Data
