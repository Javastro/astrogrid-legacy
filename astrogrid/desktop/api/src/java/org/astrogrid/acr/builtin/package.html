Access components of the Astro Runtime API

The {@link org.astrogrid.acr.builtin.ACR} class provides access to all the ciomponents available in the AR API.

{@link org.astrogrid.acr.builtin.Shutdown} allows a client of the Astro Runtime to shut down a running AR daemon.

This package also contains descriptor objects for the modules, services and methods that make up the AR API. These descrptions can be accessed from the {@link org.astrogrid.acr.system.ApiHelp} component.
