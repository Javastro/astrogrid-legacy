Access components of the Astro Runtime

Start by getting an instance of {@link org.astrogrid.acr.builtin.ACR}. This class provides access to all the services available in the Astro Runtime.

{@link org.astrogrid.acr.builtin.Shutdown} allows a client of the Astro Runtime to shut down a running AR daemon.

This package also contains datastructures to describe the modules, services and methods that are available from the Astro Runtime. This data can be accessed from the {@link org.astrogrid.acr.system.ApiHelp} component. @serial exclude
