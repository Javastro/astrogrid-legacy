The for element expresses a for loop.
The structure of the for loop is similar to the for in Python (or for-each in Javascript)
- it iterates over a sequence, rather than using an arithmetic expression like in Java / C / C++.
The for element has two required attributes: items which must evaluate to a list of items to iterate over;
and var which provides the name of the loop variable to assign each element of the list to.
The body of the for element is an activity that will be executed for each item on the list.
