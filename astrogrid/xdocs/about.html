How to change these documents

These documents are stored in the Astrogrid CVS repository under astrogrid/xdocs. To add a document simply drop a simple html file or xdoc into this folder.

They are built and deployed automatically by the scripts in astrogrid/mavenrun. The version stamp at the top of this page indicates which build the documentation corresponds to, and the time it was built. For new merge enquiries, please add an entry into changes.xml.
