Certificate-chain handling according to RFC3820. This package handles certificate chains that may include proxy credentials.

The classes in this package support the security facade but are not part of the public API. They should not be used directly by applications.

The sub-packages tomcat5 and tomcat6 contain RFC3820-enabled implementations of SSL for Apache Tomcat, versions 5.5 and 6.0 respectively. Due to structural issues in Tomcat, the sub-packages replicate classes from this current package.
