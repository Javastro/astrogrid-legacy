AstroGrid Portal Installer

Version @PORTAL-VERSION@

Introduction


This software will download and install the AstroGrid Portal on a local or remote server.
The Portal is an html-based web front end offering access to many AstroGrid services. For information see the portal web pages at http://www.astrogrid.org/maven/HEAD/portal.
Download more AstroGrid software from http://software.astrogrid.org

Prerequisites

  1. A connection to the Internet
  2. A target machine running Tomcat 5.0 or higher, with the Manager webapp enabled
  3. Username and password for the Manager webapp
  4. The end-point of a live AstroGrid Registry (for example: http://hydra.star.le.ac.uk:8080/astrogrid-registry/services/RegistryQuery )
  5. Java jdk 1.4 (probably present if you are reading this!)

Windows Users

Important: if your Tomcat server is running under Windows, the Portal will not uninstall completely. To complete the uninstallation, shut down tomcat and delete the astrogrid-portal folder from %CATALINA_HOME%/webapps,
