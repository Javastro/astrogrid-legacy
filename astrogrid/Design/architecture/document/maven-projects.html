Organization of Java source code

AstroGrid's products are built using Maven. There are many Maven "projects" which are described in the UML model and in the reports made by Maven itself (during AstroGrid's scheduled builds of the system). many Maven projects have a groups sub-projects; e.g. separate sub-projects to build client-side and server-side components.

The Java packages are dispersed among the Maven projects. Many packages are actually split between several projects. The map of code in the UML description of the projects may help to find where a given class has been stored.
