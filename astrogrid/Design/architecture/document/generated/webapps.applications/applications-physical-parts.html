Component cec-http
A small servlet that helps CEA by querying servlet context
  • Fiinds out where it is and sets keys in config from this (keys used to generate VODEscription for this CEA service).
  • If .boot.cea is set to true, invokes startup() on the pico container on activation of the web application.
  • Provides CGI access to some of the CEC methods (called via HTTP-get with no params to get usage information).
