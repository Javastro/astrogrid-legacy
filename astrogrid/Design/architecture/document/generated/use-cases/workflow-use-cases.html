UseCase Diagram workflow-use-cases
package: use-cases
All queries and data-transforming operations in AstroGrid are done via workflows. Operations that appear atomic to the end-user are managed as single-step workflows. Thus, the use cases on this diagram cover all the queries and data-processing done in AstroGrid.
