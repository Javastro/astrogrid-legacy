Collaboration Diagram get-filestore-delegate-collaboration
package: use-cases
Get a delegate object for a Filestore service identified by an IVORN. Objects inside the MySpace web-application invoke the resource registry, as a SOAP service, to get the service endpoint of the the Dilestore service. This collaboration follows the 'service resolver' pattern, but with some variations. Firstly, there are two layers of resolver instead of one. Secondly, the IVORN given to the resolver is for a file stored on the Filestore, not for the Filestore itself. A FileStoreIvornParser is used to get the service IVORN from the file IVORN.

Note:
Derives the IVORN for a Filestore from the IVORN for a file on a Filstore.
