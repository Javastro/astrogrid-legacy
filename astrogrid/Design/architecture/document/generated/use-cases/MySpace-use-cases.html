UseCase Diagram MySpace-use-cases
package: use-cases

Currently, the only explicit use-case available for MySpace is to download data into a web browser. The processes by which data are lodged in MySpace are part of the workflow use-cases (q.v.)

In the current system, MySpace is contained entirely inside AstroGrid. In future systems, MySpace will be part of VOSpace and will have external partners.
