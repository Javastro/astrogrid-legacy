Activity Display MySpace page

Portal obtains the metadata for the current display of MySpace and displays them in a tree structure. The defaultl display shows just the files and directories in the home directory. Subsequent tree-walking commands can expand some of the sub-directories. User "walks the tree" to find the appropriate file.

For each file displayed , there is a download link. Portal has to interrogate MySpace to get the target URL for the link. When the appropriate file is found, user follows that link.

Activity download file
User's browser invokes a HTTP download for the file. The contents are sent back from MySpace. The browser handles the contents according to the MIME-type in the HTTP headers.
