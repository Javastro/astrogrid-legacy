Activity Diagram determine-url-for-myspace-file
package: use-cases

Client knows of a file in MySpace and needs a URL from which to download the file contents. MySpace is an implementation of VOSpace;therefore the file is named according to VOSpace rules; therefore the file is identified by an IVORN.

Activity Resolve AGSL
Client asks the AGSL to resolve itself to a a URL. Since this file is in MySpace, the AGSL has an internal MySpace Resource Locator (MSRL). It uses this to generate a delegate for the MySpace manager holding the file and asks that manager to supply the download URL for the file.
