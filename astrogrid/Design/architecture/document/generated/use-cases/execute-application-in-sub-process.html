Execution of a command-line application in a sub-process

The CommandLineApplication object is called by the web-service wrapper. It retrieves the arguments for the application (zeroth argument is the programme name) from the List prepared when it, the CommandLineApplication, was constructed. It starts the application in a sub-process, then spawns a thread to watch while the sub-process completes.

When spawning the thread, the CommandLineApplication specifies itself as the runnable object.
