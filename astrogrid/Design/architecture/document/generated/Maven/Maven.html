package: Maven

Arrangement of the code into Maven projects

AstroGrid code is organized into Maven "projects". This package tree lists those projects, their products and their compile-time dependencies.

Some of the Maven projects map to deployable sub-systems. The rest generate compile-time artefacts that are included in other projects to make the deployable sub-systems.

Most of the Maven projects contain Java source-code. The Maven packages modelled here include links to the corresponding Java packages inside the model; the links are listed at the end of the description of each Maven package. To see Maven's own Javadocs (produced in AstroGrid's scheduled builds of the systems), please follow the hyperlink to the project's Maven report.
