Class Diagram Manager-service-interfaces
package: webapps.myspace.Manager
MySpace Manager has both compile-time and run-time WSDL contracts. The compile-time contract is derived from the Java interface defining the service. For modelling purposes, the contract is a trivial extension of the interface. The run-time contract is derived, by Axis at run time, from the implementation class. The latter is derived, using Axis tools at compile time, from the compile-time contract. The two contracts are equivalent in terms of operations, but use different names for their major elements. The implementation class derived by Axis delegates all methods of the interface to a hand-coded implementation class. Note that both these classes are actually in the package org.astrogrid.store (the client package), not org.astrogrid.mySpace.
