Interface Summary
FitsReader An interface so that appilcations that want to read FITS files can do so no matter where they are.
Class Summary
FitsData Abstract class representing the data part of the HDU (Header Data Unit).
FitsFormatException An exception for generation by fits file readers

The original exception should wherever possible be preserved as the cause
FitsHdu Represents a 'header/data unit' which may be the original 'primary' hdu at the beginning of a fits file, or an extension that also consists of a header/data pairing.
FitsHeader A Fits header is the ascii 80-bytes keyword/value pair lines of the primary hdu (header/data unit) or extension unit.
FitsKeyword Represents a keyword/value pair in the header data unit (hdu) of a fits file.
FitsStreamReader A set of routines for reading a fits image

FITS images are binary representations of tables or pixel images, and are well known in the astronomical community (in 2003!).
FitsWCS Calculates World COordinates from the given FITS header.
