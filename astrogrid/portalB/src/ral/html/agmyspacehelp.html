MySpace Browser Help

MySpace Explore will show all the files stored in MySpace when the page is first loaded.



Function Supported:

  • [New folder]: This option will take you to Create New Folder Page. Please fill in the full path name of the folder you wish to create. eg. /clq/serv1/folder1 Please make sure that all folders will have to be under /userName/serv1/ or /userName/serv2.

  • [Cut]: Click on this Link if you wish to Cut a file. This option only support file transferring, not folders.

  • [Copy]: Click on this Link if you wish to Copy a file. This option only support file transferring, not folders.

  • [Paste]: This option is hidden unless you have Clicked on Copy or Cut, in which case, the page will refresh and display this option against all possible locations. Click on Paste next to the location you want to transfer your file to. This option only support file transferring, not folders.

  • [Rename]: This option will take you to Rename File page. Please make sure all information displayed on this page are correct and fill in the file name including full path you wish to rename to. This option only support file transferring, not folders.

  • [View]: This option is hidden unless you have clicked on one of the files in your file tree or one of the options(Copy,Cut,Rename...) next to the file. View will take you to another page which displays your file(in this iteration we assuming all files are VOTables).

  • [Delete]: This option will delete a file or an empty folder. Notice that Serv1 and Serv2 are not allowed to be deleted.



    Trouble shooting: Please contact AstroGrid for any unexpected errors.
