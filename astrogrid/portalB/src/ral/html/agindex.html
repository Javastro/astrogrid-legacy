Welcome



Welcome to the AstroGrid Virtual Observatory Portal


This portal is running the latest version of the AstroGrid software, Iteration 2.


Iteration 2 is a foundation release, with great emphasis placed upon the infrastructure design. It is the platform upon which the rest of AstroGrid will be built and as such has been designed with maximum flexibility and scalability in mind. We hope you enjoy using this Portal. Come back frequently to see the progress being made.


AstroGrid is a £3.7M project aimed at building a data-grid for UK astronomy, which will form the UK contribution to a global Virtual Observatory. More information can be found at the AstroGrid home  and Wiki pages.
