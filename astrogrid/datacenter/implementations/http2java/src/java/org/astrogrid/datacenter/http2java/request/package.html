Code to call legacy web service method

This package contains implementations of {@link org.astrogrid.datacenter.legacy.RequestMapper} for the HTTP-GET and HTTP-POST protocols.

These implementations use the commons-httpclient library to perform the communication. The commonality between the implementations is captured in the {@link org.astrogrid.datacenter.legacy.AbstractHttpRequest} class.

Other protocols / transport methods could be implemented and then specified in the xml web service descriptor using the 'class-name' attribute.
