See package {@link org.astrogrid.datacenter.delegate} for documentation of how to call the datacenter web service programatically.

See package {@link org.astrogrid.datacenter.tools} for documentation of how to call a datacenter web service using a command-line tool.

The query languages currently supported are ADQL (see {@link org.astrogrid.datacenter.adql}) and SQL (see {@link org.astrogrid.datacenter.sql}).
