SOAP-y Beans

This package contains data beans that are transported via SOAP between client and server. The sources are generated from the wsdl and xsd files by the wsdl2java tool.

Most important is the {@link org.astrogrid.datacenter.axisdataserver.types.Query} class, which encapuslates a query to the server. The class {@link org.astrogrid.datacenter.axisdataserver.types.QueryHelper} provides static methods to assist in building Query objects.
