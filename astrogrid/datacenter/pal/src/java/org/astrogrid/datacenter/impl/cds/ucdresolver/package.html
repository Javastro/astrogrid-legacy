Delegate for Sesame Name Resolver Web service

Delegate class {@link org.astrogrid.datacenter.cdsdelegate.sesame.SesameDelegate} provides access to this service. NB at time of writing the service doesn't appear to work.

The Deprecated web methods of the service do work however. These are wrapped by the {@link org.astrogrid.datacenter.cdsdelegate.sesame.DeprecatedSesameDelegate}

Service is documented at http://cdsweb.u-strasbg.fr/cdsws/name_resolver.gml
