Introduction


This software will download and install the AstroGrid FileStore component.
FileStore and FileManager together make the AstroGrid MySpace distributed storage system - for information on this and other AstroGrid components see the software site http://software.astrogrid.org.

Prerequisites

  1. A connection to the Internet
  2. A target machine running Tomcat 5.0 or higher, with the Manager webapp enabled
  3. Username and password for the Manager webapp
  4. (Optional) An AstroGrid Registry (for example: http://hydra.star.le.ac.uk:8080/astrogrid-registry )
  5. Java jdk 1.4 (probably present if you are reading this!)

Installation

This installer will guide you through the installation process, downloading any necessary software and prompting you for configuration options. If you have previously installed FileStore, you can use the installer to remove this instance, reinstall it, or register it. If you are reinstalling FileStore, you may wish to preserve an existing file repository by unchecking the "Initialise FileStore Repository" box.
Important Note
If you are installing a FileStore that is to be used and registered externally to the target machine you MUST replace the Tomcat URL of "http://127.0.0.1:8080" with the full externally visible URL when prompted.
