AstroGrid MySpace

This site is an AstroGrid MySpace service. Warning - the following docs are out of date.

From here you can:

  • Browse the Maven documentation for this release,
  • Check HappyAxis to see if Axis has installed successfully, or
  • Check HappyMySpace to see if MySpace has installed successfully and configure it, or
  • See what functions are available.
  • Browse the files
  • View the wsdl for the web services.

To install the latest MySpace go to the Maven documentation and follow the links on the left.

Several papers describing the MySpace service have been published or are in preparation. On-line versions are available which can be consulted for background information.
