MySpace Service Test Pages

These pages allow you to test a few of the functions of this MySpace service. The functions cannot be used in an arbitrary order. It is, for example, necessary to create at least one user of the service before you use any of the other functions. Similarly, you must create a container before you can put a file in that container (though some standard containers are created automatically when you create a user). It is necessary to import a file into the MySpace service before you can copy or export a file.

User `clq' belonging to AstroGrid community `lei' is used as the default in the example pages. You can, of course, substitute your own values if you wish.
