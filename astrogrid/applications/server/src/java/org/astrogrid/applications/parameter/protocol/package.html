Reading and Writing External values

This package contains code that enables external values (e.g. 'indirect' parameters -- whose values are stored in some external resource) to be read and written via a variety of protocols. The package defines a library for handlers, and handlers for common protocols.

Start with a {@link org.astrogrid.applications.parameter.protocol.DefaultProtocolLibraryFactory} and use this to create an instance of {@link org.astrogrid.applications.parameter.protocol.ProtocolLibrary}. From this, {@link org.astrogrid.applications.parameter.protocol.ExternalValue} instances can be manufactured from URI or String references. ExternalValue instances provide methods to read / write to that location via streams.
