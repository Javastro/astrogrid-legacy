Descriptions of Applications - parameters, interfaces, etc.

This package is concerned with the description of the applications that are provided by the server.

The overall description of the application is defined by the {@link org.astrogrid.applications.description.ApplicationDescription} class, which has associated {@link org.astrogrid.applications.description.ParameterDescription} and {@link org.astrogrid.applications.description.ApplicationInterface}.

The descriptions are contained within a {@link org.astrogrid.appliations.description.ApplicationDescriptionLibrary}
