Introduction


This software will download and install the AstroGrid Job Entry System component.
For information on this and other AstroGrid components see the software site http://software.astrogrid.org.

Prerequisites

  1. A connection to the Internet
  2. A target machine running Tomcat 5.0 or higher, with the Manager webapp enabled
  3. Username and password for the Manager webapp
  4. An AstroGrid Registry (for example: http://hydra.star.le.ac.uk:8080/astrogrid-registry )
  5. Java jdk 1.4 (probably present if you are reading this!)

Installation

This installer will guide you through the installation process, downloading any necessary software and prompting you for configuration options. If you have previously installed JES, you can use the installer to remove this instance or reinstall it. During installation you will be prompted for a location JES can use as its working directory - this must be writable by Tomcat (and JES given permission under any security manager you may be running).

Listeners, proxies and URLS - important

When you enter the URL of your local Tomcat ensure that you use the full external URL of the machine, ie not 127.0.0.1. This is important as it is used to construct the URLs of the JES listeners that are set in the webapp's JNDI properties.
Note - you can override the values of these listeners by setting the properties
jes.monitor.endpoint.url and
jes.results.endpoint.url
in your properties input file. These will take precedence over any urls calculated by the installer. This is particularly useful if you are using a proxying service.
