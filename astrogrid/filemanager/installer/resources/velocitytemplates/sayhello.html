Introduction


This software will install the AstroGrid FileManager component.
FileManager and FileStore together make the AstroGrid MySpace distributed storage system - for information on this and other AstroGrid components see the software site http://software.astrogrid.org.

Prerequisites

  1. A target machine running Tomcat 5.0 or higher, with the Manager webapp enabled
  2. Username and password for the Manager webapp
  3. (Optional) An AstroGrid Registry (for example: http://hydra.star.le.ac.uk:8080/astrogrid-registry )
  4. Java jdk 1.4 (probably present if you are reading this!)

Installation

This installer will guide you through the installation process, downloading any necessary software and prompting you for configuration options. If you have previously installed filemanager, you can use the installer to remove this instance, reinstall it, or register it.
Important Note
If you are installing a FileManager that is to be used and registered externally to the target machine you MUST replace the Tomcat URL of "http://127.0.0.1:8080" with the full externally visible URL when prompted.
