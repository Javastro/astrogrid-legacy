Public interface to the filemanager

This package contains the public interface to the filemanager client.
  • Start by instantiating a {@link org.astrogrid.filemanager.client.FileManagerClientFactory}.
  • Log in from this to get a {@link org.astrogrid.filemanager.client.FileManagerClient}.
  • From the client, the home space of the user, or other arbitrary file manager locations, can be accessed. The {@link org.astrogrid.filemanager.client.FileManagerNode} represents a node (i.e. a file or folder) in the filemanager
