Implementation of the main class filemanager server

{@link org.astrogrid.filemanager.server.TheFileManager} is the production version. {@link org.astrogrid.filemanager.server.TestFileManager} is the testing version - which uses mocks where needed.

Each are constructed by assembling components to produce a {@link org.astrogrid.filemanager.server.CoreFileManager}, and then wrapping this in decorators that add additional functionaltiy.
