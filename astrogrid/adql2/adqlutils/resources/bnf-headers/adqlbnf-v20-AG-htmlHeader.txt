This file contains a BNF definition of ADQL, the Astronomical Data Query Language, 
at the level of AstroGrid's first projected support of v2.0 of ADQL.
The elements are arranged in alphabetical order. 
ADQL is based upon a subset of SQL92 with extensions for astronomical usage.
The select statement is found at <a href="#query_specification">&lt;query_specification&gt;</a>.

See <a href="http://savage.net.au/SQL/sql-92.bnf.html">SQL92</a> for a similar page describing SQL92 in full.
