AstroGrid Registry Index

Normal operation of the registry:

  • Setup a new Registry (see below)
  • Query Registry via ADQL
  • Query Registry for a specefic Identifier
  • List All Registry types
  • List All items (only 25 usefull for small registries)
  • Verify Axis installation okay

For initial set-up of the registry see the set-up page (password protected) to the servlet containers "manager" role.
