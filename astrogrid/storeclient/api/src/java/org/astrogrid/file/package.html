File API

A package to provide a single straightforward access to filesystems. It includes an interface definition and implementations for File and ZipFile, for accessing local filesystems.

NB It is not meant to provide fancy stuff like threaded updates or monitoring; it provides only a common interface for a variety of access methods. (Implementations will probably need to include 'Principal's in the file constructors for secure access).
