Quaestor @VERSION@, @RELEASEDATE@

The Quaestor service is a reasonably handy triplestore and SPARQL endpoint, with a RESTful interface. It's built on top of Jena.

The following files are available for download:

@WARFILE@
The Tomcat .war file. Drop this in a running Tomcat, or upload it to the management interface.
@JETTYZIP@
The standalone Jetty server. Unzip this, and start the server with the start-jetty.sh shell script.
@CLIENTZIP@
API docs, and a small jar file which includes a REPL which allows you to talk to the server if you want to (I imagine rather few folk really want either, so this is I'm afraid barely documented)

The source code is in the AstroGrid project's CVS repository. That should be publicly available -- get in touch with me if you have difficulty getting at it.

The Quaestor server's interface is documented here, as well as on the front page of the running Quaestor service.
