Quaestor is a Tomcat servlet providing a generic SPARQL endpoint.

It allows clients to upload multiple sets of RDF assertions, and then perform SPARQL queries on them. It has both a RESTful interface and an XML-RPC one.

For details of the interface, see the top page of the running servlet, or web/index.html in the source code distribution.

It is implemented using the Jena library from HP.
