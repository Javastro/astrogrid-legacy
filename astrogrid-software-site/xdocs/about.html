How to change these documents

These documents are stored in the AstroGrid CVS repository under /astrogrid-software-site/xdocs. To add a document simply drop a simple html file or xdoc into this folder. They can be regenerated using maven by typing maven astrogrid-deploy-site from the astrogrid-software-site directory.

To generate the main site docs for each release, 1. rtag the whole astrogrid with a release number e.g r0_10_0 2. make certain changes in astrogrid-software-site and astrogrid/xdocs to include the release link 3. on the build machine, make sure the DOCLOCATION is not set to HEAD e.g: "export DOCLOCATION=/var/www/www/maven/docs/0.10.0". 4. on the build machine, run "publish-component.sh (component name) tag_number" 5. on build machien, run "publish-release-docs.sh r0_10_0" to publish the main page of this release
