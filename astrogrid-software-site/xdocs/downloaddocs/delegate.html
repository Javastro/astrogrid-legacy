Software for third party developers

Most AstroGrid components are called via webservices and third parties are encouraged to develop applications that interoperate with them. To facilitate this AstroGrid provides a number of delegate/client jars that enables Java applications to call AstroGrid components transparently, without having to deal with the web services directly.

Follow the links to the left to download the most recently built delegate/client jars for each component. Alternatively, list all of the latest AstroGrid delegate jars, or the last stable release.
