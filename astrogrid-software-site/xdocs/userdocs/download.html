Downloading Astrogrid Software

Binaries

Precompiled binaries may be downloaded from our repository. The Quick Downloads link to the left offers shortcuts to speedily find the component and release you want. Altenatively, if you are using AGINAB to install Astrogrid, the correct components will be automatically downloaded during the installation process.

Source code

AstroGrid source code is freely downloadable from our CVS repository.
