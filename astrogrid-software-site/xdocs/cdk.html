Core Client Development Kit

The Astrogrid Core CDK is for developers that wish to interact with the core Astrogrid services (Myspace / VoSpace, Registry, Community). It contains all the required delegate jars, plus their dependencies, example configuration, and links to online documentation

The Core CDK is available for download ( .zip, .tar.gz) and also as a JNLP Extension that can be referenced by webstart applications.

Workflow Client Development Kit

The Astrogrid Workflow CDK is for developers that wish to interact with the Astrogrid workflow services (Jes, CEA). It also contains the delegates for accessing the core Astrogrid services (Myspace / VoSpace, Registry, Community). (which are also available separately in the smaller Core CDK). It contains all the required delegate jars, plus their dependencies, example configuration, and links to online documentation

The Workflow CDK is available for download ( .zip, .tar.gz) and also as a JNLP Extension that can be referenced by webstart applications.
